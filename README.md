# C++ File and Date/Time Information Example Programs

The programs are as follows:

|Program Description                                                  |Sub-folder     |
|---------------------------------------------------------------------|---------------|
|Outputs the current date and time.                                   |CurrDateTime   |
|Outputs information (e.g last modified time) about one or more files.|PrintFileInfo  |
|Compares the last modified time of 2 files.                          |CmpLastModified|

For more information about the programs see the README files in the
above-mentioned sub-folders.

To create the executables for the programs use the **Makefile** in the
**build** sub-folder. See the README file in the **build** sub-folder for more
information.
